# Web Map Bouaké - PFE Mastère Sciences Géomatiques

Projet de web mapping informatif de l'échangeur projeté au carrefour Bouaké.

## Contenu

- `index.html` : page d'accueil et carte interactive.
- `pfe-custom.css` : styles personnalisés de la page d'accueil, de la carte et de la légende.
- `pfe-custom.js` : logique personnalisée pour la légende des lignes de parcours.
- `images/` : logos et image de l'échangeur.
- `layers/`, `styles/`, `resources/`, `webfonts/` : fichiers générés par qgis2web.

## Hébergement avec GitHub Pages

1. Créer un dépôt GitHub public, par exemple `web-map-bouake`.
2. Déposer directement tout le contenu de ce dossier à la racine du dépôt.
3. Aller dans `Settings > Pages`.
4. Choisir `Deploy from a branch`.
5. Sélectionner la branche `main` et le dossier `/ root`.
6. Enregistrer.

Le site sera disponible à une adresse du type :

`https://votre-utilisateur.github.io/web-map-bouake/`

## Remarque

Le fond OpenStreetMap nécessite une connexion Internet.
